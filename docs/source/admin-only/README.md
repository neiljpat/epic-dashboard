# Admin-only documents — DO NOT SHARE

**Nothing in this folder should ever be uploaded to Google Drive, sent to members, or shared in any way.**

These documents contain PII (Social Security Numbers, addresses, personal allocations) or are otherwise scoped to a single member or to the admin only.

## What goes here

### `el-pen-llc/`
- El Pen LLC operating agreement (contains member ownership %)
- El Pen LLC K-1s to individual members (each member's K-1 from the LLC, contains SSN, address, individual allocation %)
- El Pen LLC tax returns
- El Pen LLC bank statements

### `personal/<member>/`
- Personal investment documents owned by an individual member
- Examples: Neil's personal Cherish Health SAFE docs, Dan's personal Ensemble VC II subscription agreement, AW1 Q1 statements
- These belong to the owning member, not the club. Keep them filed but do not share with other members.

### `subscription-agreements/`
- LP subscription agreements signed by El Pen LLC for each fund
- Contains signatures, banking info, ownership disclosures

### `side-letters/`
- Side letter agreements with GPs (custom terms)
- Confidential by their nature

### `wire-confirmations/`
- Wire transfer confirmations
- Bank routing info, account details

## If you receive an El Pen LLC K-1 from your accountant

The accountant will issue **one K-1 per member** showing each member's pro-rata allocation of EPIC's gains/losses. Each of those K-1s contains the member's:
- Social Security Number (or substitute)
- Home address
- Individual capital account balance
- Allocated share of fund-level items

These files go in `el-pen-llc/<tax-year>/k1-<member-name>.pdf` and never leave this folder.
